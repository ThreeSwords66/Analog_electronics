## 电气工程和电子信息技术的区别  

**电工用于解决电气设备的，无线通信设备，早期的电子信息工程用于处理电子信息，电气工程用于处理发电，输电，用电。简单来说就是围绕电能展开**，但是电气工程及其自动化，自动化控制早期使用机械结构自动控制，抽水马桶等等。但是当计算机出现的时候，[^1]自动化飞快的发展。  
总的来说  

- 电气工程  围绕电能
- 电子信息  围绕电子信息
